# テスト実行用

## Installation

* macOS
  > $ brew install k6
* docker
  > $ docker pull loadimpact/k6

## Execution

* macOS
  > $ k6 run index.js
* docker
  > $ docker run -i loadimpact/k6 run - <script.js>
